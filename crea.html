About This Project



As I keep on making iOS app, I noticed that I'm always writing same code or some pattern that appears so often. I thought that now is the time to make my own framework. I put a lot of useful extension so that I can write a code in a type-safe way and more efficiently. The most remarkable characteristics is that all methods in this library are invoked through a special property "crea", that is, I follow the notation that RxSwift and KingFisher does. Through this project, I've been learning how to make iOS framework, maintain one project. At the same time, I think this project also has meaning that it shows my level in Swift.
